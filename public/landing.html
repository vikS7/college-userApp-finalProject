Smart Reflect is a modular, extensible and open source smart mirror platform made with idea of being widly available to eveyone. Our platform provides simple and effective way to increase the functionality of already existing smart mirror platforms. We designed and developed SmartReflect—a smart mirror platform that offers two main benefits, first it is modular and extensible. Developers can add plugins to customize their smart mirror applications and second it utilizes a server design that allows the developer to sidestep a sandbox created by a web browser.

Being an open-source development it allows an abundance of applications to be developed by multiple people, expanding this project beyond what one team can easily achieve
